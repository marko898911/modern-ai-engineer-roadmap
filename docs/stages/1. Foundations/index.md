# Stage 1: Foundations

<span class="stage-badge">1</span> Become comfortable enough with programming, math, and data to learn AI without fighting your tools.

## Goal

You do not need a full computer science degree before touching AI. You do need
enough foundation to debug, read code, reason about data, and understand model
behavior.

## Learn

| Area | Minimum useful depth |
|---|---|
| Python | functions, classes, virtual environments, packages, type hints, tests |
| Git and terminal | clone, branch, commit, diff, shell navigation, environment variables |
| Data stack | NumPy, pandas, matplotlib, JSON, CSV, Parquet basics |
| Math | linear algebra, probability, gradients, statistics, logarithms |
| CS basics | complexity, memory, HTTP, APIs, databases, concurrency basics |
| Software practice | logging, testing, config files, reproducible scripts |

## Build

Build a small data application:

- load a real dataset
- clean it
- visualize 5 useful facts
- expose one result through a tiny API or CLI
- write tests for at least one data function

## Measure

Measure:

- data rows before and after cleaning
- missing values
- runtime of the main script
- one correctness check

## Exit Criteria

You are ready for Stage 2 when you can:

- create and activate a Python environment
- install dependencies and run tests
- load, clean, and visualize a dataset
- explain vectors, matrices, dot products, probability, and train/test split
- use Git to save your work cleanly

## Common Mistake

Do not spend months collecting math resources. Learn the math that explains the
model you are currently building, then return for deeper math when the model
demands it.
