# Tokenization

## Purpose

Explain how text becomes tokens before the model reads it.

## Contributor Scope

Cover:

- tokens versus words
- subword tokenization
- why code, JSON, and non-English text may tokenize differently
- token counting for prompts, tool schemas, retrieved context, and answers

## Build

Use a tokenizer to compare a normal sentence, code snippet, JSON object, and
multilingual sentence.

## Exit Criteria

You can estimate why a prompt became expensive or too long.
