# Agent Loop

## Purpose

Show how an AI agent moves from a user goal to actions and observations.

## Contributor Scope

| Step | Meaning |
|---|---|
| Perception | receive user input, files, events, or environment state |
| Plan | decide what information or action is needed next |
| Action | call a tool or produce a structured output |
| Observation | read tool results and update state |
| Reflection | check progress and recover from mistakes |
| Stop condition | finish, ask for help, or fail safely |

## Build

Draw one loop for a research assistant and one loop for a code assistant.

## Exit Criteria

You can explain why an agent loop needs max steps, timeouts, and stop rules.
