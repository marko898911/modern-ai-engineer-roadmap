# Stopping Criteria and Max Length

## Purpose

Teach output limits so agent responses stay bounded, parseable, and affordable.

## Contributor Scope

| Topic | What to explain |
|---|---|
| Stop sequence | text pattern that ends generation |
| Max length | maximum output tokens |
| Truncation | answer or JSON can be cut too early |
| Output budget | enough space for the expected result without wasting tokens |

## Build

Test one JSON task with too-small, reasonable, and too-large output limits.

## Exit Criteria

You can set output limits without breaking the expected response format.
