# Understand the Basics

## Purpose

Connect the prerequisite topics into one beginner mental model before entering
LLM fundamentals.

## Contributor Scope

Create a short checklist:

- What is the model?
- What input does it receive?
- What output does it generate?
- What data can it see?
- What tool can it call?
- What should it never do?

## Build

Describe a tiny study-helper agent in one paragraph and label the model,
prompt, context, tool, output, and safety boundary.

## Exit Criteria

You can read a simple agent design and identify its moving parts.
