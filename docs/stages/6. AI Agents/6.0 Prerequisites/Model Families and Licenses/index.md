# Model Families and Licenses

## Purpose

Help learners choose models responsibly by understanding model families,
providers, weights, and licenses.

## Contributor Scope

| Topic | What to explain |
|---|---|
| Model family | related models from the same provider or project |
| Closed-weight model | model used through an API or hosted service |
| Open-weight model | model weights are available under a license |
| License | rules that define allowed use, redistribution, and commercial rights |
| Model card | summary of training, limits, benchmarks, and intended use |

## Build

Compare two model families and write down provider, license, context length,
tool support, and expected use case.

## Exit Criteria

You can tell whether a model is suitable for a student project, private system,
or public product.
