# 6.0 Prerequisites

## Goal

Before building agents, learn the vocabulary and mental model that make LLM
and agent behavior understandable. This sub-stage is intentionally lightweight:
it prepares students to enter LLM fundamentals without drowning in advanced
details.

## Parts

| Part | What contributors should cover |
|---|---|
| [Basic Knowledge](<Basic Knowledge/index.md>) | AI, ML, deep learning, LLM, prompt, token, tool, agent |
| [Transformer Models and LLMs](<Transformer Models and LLMs/index.md>) | why transformers power modern LLMs |
| [Model Families and Licenses](<Model Families and Licenses/index.md>) | open-weight vs closed models and license basics |
| [Understand the Basics](<Understand the Basics/index.md>) | connect all prerequisite ideas with a small study checklist |

## Manager Notes

Use this page to control the sub-stage scope. Add, remove, or rename child
folders only when the navigation in `mkdocs.yml` is updated at the same time.

## Exit Criteria

Move on when you can:

- explain what an LLM is in beginner language
- explain why prompts, tokens, context, and tools matter for agents
- tell the difference between open-weight and closed-weight models
- read a model card or license summary without getting lost
