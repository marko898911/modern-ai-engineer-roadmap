# Basic Knowledge

## Purpose

Give beginners the vocabulary they need before studying LLMs and agents.

## Contributor Scope

| Concept | Beginner meaning |
|---|---|
| AI | software that performs tasks usually associated with intelligence |
| Machine learning | systems that learn patterns from data |
| Deep learning | machine learning with neural networks |
| LLM | a language model trained to predict and generate text tokens |
| Prompt | the instruction and context sent to a model |
| Agent | a model-driven loop that can choose actions and use tools |
| Tool | external capability the model can request, such as search or code execution |

## Build

Create a one-page glossary and explain each term with one example.

## Exit Criteria

You can explain the difference between a model, a prompt, a tool, and an agent.
