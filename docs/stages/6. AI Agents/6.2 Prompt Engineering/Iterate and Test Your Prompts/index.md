# Iterate and Test Your Prompts

## Purpose

Teach prompt improvement as an evaluation process, not a guessing game.

## Contributor Scope

Cover:

- prompt versions
- fixed test cases
- normal, edge, and adversarial tests
- regression tracking
- one-change-at-a-time iteration

## Build

Create a 10-case prompt test set and record the result before and after one
prompt change.

## Exit Criteria

You can prove whether a prompt change improved the system.
