# Specify Length, Format, etc

## Purpose

Teach output constraints that make answers useful for users and safe for
downstream parsers.

## Contributor Scope

Cover:

- length
- format
- JSON schema
- markdown style
- citation rules
- refusal or escalation format
- parser validation

## Build

Write one free-form prompt and one schema-constrained prompt. Compare how easy
they are to parse.

## Exit Criteria

You can design an output format and safely reject malformed output.
