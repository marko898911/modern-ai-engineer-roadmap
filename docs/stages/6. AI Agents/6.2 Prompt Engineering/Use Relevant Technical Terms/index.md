# Use Relevant Technical Terms

## Purpose

Teach precise language so the model, developer, and evaluator mean the same
thing.

## Contributor Scope

Include examples such as:

- `return JSON matching this schema`
- `cite retrieved sources`
- `ask a clarifying question`
- `draft but do not send`
- `use this tool only when`

## Build

Take five vague phrases and replace them with technical instructions.

## Exit Criteria

You can define every technical term used in the prompt.
