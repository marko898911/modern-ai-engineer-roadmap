# Be Specific in What You Want

## Purpose

Teach learners to replace vague requests with clear tasks, boundaries, and
success criteria.

## Contributor Scope

Explain:

- task
- audience
- output quality bar
- allowed and forbidden actions
- when the agent should stop

## Build

Rewrite five vague prompts into specific prompts.

## Exit Criteria

You can tell whether a prompt has a measurable success condition.
