# Overview

## Purpose

Explain what prompt engineering means when the model is part of an agent loop.

## Contributor Scope

Cover:

- role and goal
- task context
- constraints and permissions
- tool policy
- output format
- recovery behavior
- examples and eval cases

## Build

Write one agent prompt with role, goal, context, tool policy, safety rule, and
output contract.

## Exit Criteria

You can explain every instruction in your prompt.
