# Prompting Techniques

## Contributor Scope

Cover the roadmap.sh prompting technique group:

- Zero-Shot Prompting
- One-Shot / Few-Shot Prompting
- System Prompting
- Role Prompting
- Contextual Prompting
- System / Role / Contextual prompting
- Step-back Prompting
- Chain of Thought (CoT) Prompting
- Self-Consistency Prompting
- Tree of Thoughts (ToT) Prompting
- ReAct Prompting

## Build

Choose three techniques and test them on the same task. Compare quality,
stability, cost, and failure modes.
