# What Is Prompt Engineering

## Contributor Scope

Explain prompt engineering as the design, testing, and versioning of model
inputs to get useful, safe, repeatable outputs.

Cover:

- prompt design
- prompt testing
- prompt versioning
- prompt failure analysis
- difference between prompting, RAG, tools, and fine-tuning

## Build

Rewrite one weak prompt into a testable prompt and explain the changes.
