# Provide Additional Context

## Purpose

Show how useful context helps the model answer correctly without guessing.

## Contributor Scope

Cover:

- user state
- retrieved documents
- tool results
- product rules
- trusted instructions versus untrusted evidence

## Build

Create one prompt with no context and one with relevant context. Compare the
answers and identify what improved.

## Exit Criteria

You can add context without burying the main instruction.
